# Miscellaneous Python Programs

This repository hosts miscellaneous Python programs I've written that others
may find or have found useful.

## How to Run

To view and optionally run the following programs online, click below:
- [Horizon Calculator](https://joshlsastro.github.io/python-online/?https://raw.githubusercontent.com/joshlsastro/misc-python-programs/refs/heads/main/Horizon_Calc.py)
- [How far away could you see an alien ship entering the Solar System?](https://joshlsastro.github.io/python-online/?https://raw.githubusercontent.com/joshlsastro/misc-python-programs/refs/heads/main/How_Far_To_See_Aliens.py)
- [Magnitude Wizard](https://joshlsastro.github.io/python-online/?https://raw.githubusercontent.com/joshlsastro/star_data_finder/refs/heads/master/magnitudewizard.py)

To run the programs offline, just use [Python 3](https://python.org) to run each program.
